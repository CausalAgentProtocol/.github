# Support

Welcome to the Causal Agent Protocol (CAP) community! If you need help, please use the following resources.

## Documentation

- **[CAP Specification](https://github.com/causalagentprotocol/spec)**: The current standard, defining the verbs, causal graph traversal semantics, and transport requirements.
- **[Python SDK README](https://github.com/causalagentprotocol/python-sdk)**: Instructions on how to build a CAP client or server in Python.

## GitHub Issues

If you've found a bug or have a feature request:
- For protocol-level discussions and spec bugs, use the [Spec Issue Tracker](https://github.com/causalagentprotocol/spec/issues).
- For SDK bugs or features, use the [Python SDK Issue Tracker](https://github.com/causalagentprotocol/python-sdk/issues).
- For the Abel reference implementation, use the [abel-cap Issue Tracker](https://github.com/causalagentprotocol/abel-cap/issues).

## Community Discussions

For general questions, implementation help, or showing off what you've built, please use **GitHub Discussions** in the `spec` repository (or whatever central repo the community currently uses for discussions).
