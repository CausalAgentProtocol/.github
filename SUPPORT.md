# Support

Welcome to the Causal Agent Protocol (CAP) community! If you need help, please use the following resources.

## Documentation

- **[CAP Website](https://causalagentprotocol.io)**: The public docs and specification site for CAP.
- **[CAP Repository](https://github.com/CausalAgentProtocol/cap)**: The source for the specification, schemas, docs, and website.
- **[Python SDK README](https://github.com/CausalAgentProtocol/python-sdk)**: Instructions on how to build a CAP client or server in Python.

## GitHub Issues

If you've found a bug or have a feature request:
- For protocol-level discussions and CAP repo bugs, use the [CAP Issue Tracker](https://github.com/CausalAgentProtocol/cap/issues).
- For SDK bugs or features, use the [Python SDK Issue Tracker](https://github.com/CausalAgentProtocol/python-sdk/issues).
- For the reference implementation, use the [cap-reference Issue Tracker](https://github.com/CausalAgentProtocol/cap-reference/issues).

## Community Discussions

For general questions, implementation help, or showing off what you've built, please use **GitHub Discussions** in the `cap` repository (or whatever central repo the community currently uses for discussions).

## Direct Contact

For maintainer contact, ecosystem coordination, or developer-group questions that do not fit a public issue or discussion, email `cap@abel.ai`.

For security vulnerabilities, use `security@abel.ai` instead of the general support address.
